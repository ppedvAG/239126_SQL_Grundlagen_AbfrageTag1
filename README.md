# SQL Grundlage Abfragesprache Tag 1:

### Tag 1 Inhalt:
* Select Statements
```
SELECT * FROM Customers
```
* DISTINCT
```
SELECT DISTINCT Country FROM Customers
```
* Order By
```
SELECT Freight FROM Orders
ORDER BY Freight DESC
```
* WHERE
```
SELECT * FROM Customers
WHERE Country = 'Germany'
```
>[!NOTE]
> Vergleichsoperatoren
> (> größer als)
> (>= größer gleich)
> (< kleiner gleich)
> (<= kleiner gleich)
> = exakte Suche
> != ungleich
> <> ungleich
